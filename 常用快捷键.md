# 常用快捷键

* ⌘ - Command
* ⌃ - Control
* ⇧ - Shift
* ⌥ - Option
* ↩︎ - Return/Enter
* ⌫ - Delete
* ⌦ - 向前删除键\(Fn+Delete\)
* ⎋ - Escape \(Esc\)
* ⇥ - 右制表符\(Tab键\)
* ⇤ - 左制表符\(Shift+Tab\)

---

* 按名称查找动作行为 - ⇧⌘A
* 显示可用的意图操作的列表 - ⌥⏎
* 视图切换\(项目,结构等\) - ⌥F1\(需要fn\)
* 工具窗口和编辑器中打开的文件之间进行切换 - ⌃⇥\(加⇧键反向切换\)
* 显示导航栏 - ⌘↑
