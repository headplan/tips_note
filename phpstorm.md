# PHPStorm

##### 官方文档

https://www.jetbrains.com/help/phpstorm/2016.3/meet-phpstorm.html

#### 系统需求

* 最低1G内存,建议2G内存

* 屏幕最小分辨率1024x768

> 不需要安装JRE,因为已经捆绑了JAVA
>
> 仅支持64位系统

#### 软件安装

Windows和OS X系统正常安装即可.

##### Linux系统

下载压缩包`PhpStorm-*.tar.gz`,并解压

```
sudo tar xf PhpStorm-*.tar.gz -C /opt/
```

切换到bin目录

```
cd opt/PhpStorm-*/bin
# Run脚本安装
./phpstorm.sh
```
